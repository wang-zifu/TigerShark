# Installation

    $ git clone https://github.com/s1l3nt78/TigerShark.git
    $ cd TigerShark
    $ chmod +x install.sh
    $ sudo ./install.sh
    
    The install script will allow you the option to use TigerShark from anywhere 
    in your shell or just from the install directory. (/opt/TigerShark)
